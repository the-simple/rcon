# rcon
RCON library and client

# Installation
```
cargo install --git https://github.com/the-simple/rcon
```

# Usage
```
rcon --help             
RCON client 0.1.0
Aleksandr Lykhouzov <lykhouzov@gmail.com>
Simple RCON client

USAGE:
    rcon [FLAGS] [OPTIONS] --host <HOST>

FLAGS:
        --help       Prints help information
    -v               Sets the level of verbosity
    -V, --version    Prints version information

OPTIONS:
    -h, --host <HOST>    Sets a host
    -P, --port <PORT>    Set a port
```
